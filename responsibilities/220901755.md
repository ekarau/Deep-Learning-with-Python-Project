## Model Architecture

I designed `DeepMLP`: `784 → 128 → 128 → 128 → 128 → 10` with 4 hidden layers.

**Why 4 layers:** Vanishing gradients compound multiplicatively. Sigmoid's max derivative is 0.25, so after 4 layers: $0.25^4 \approx 0.004$. A 2-layer network would hide this. 4 layers makes the effect dramatic.

**Why 128 uniform width:** Powers of 2 for GPU efficiency. Uniform width isolates activation effects from architectural variation.

**Optional BatchNorm:** When enabled, the order is Linear → BatchNorm → Activation → Dropout. BN normalizes pre-activations into the non-saturating region.

**Loss function:** `nn.CrossEntropyLoss` which computes $-\sum_i y_i \log(\text{softmax}(z)_i)$ internally with log-sum-exp for numerical stability. This is the correct pairing for multi-class with raw logit output.

## Phase 1 Experiments

All 4 models share: SGD lr=0.01, Dropout=0.2, 15 epochs, seed=42. Only the activation changes.

### Activation Functions Tested

| Function | Formula | Derivative | Issue |
|---|---|---|---|
| Sigmoid | $\sigma(x) = 1/(1+e^{-x})$ | $\sigma'(x) \leq 0.25$ | Saturates at 0 and 1 |
| Tanh | $\tanh(x)$ | $1-\tanh^2(x) \leq 1$ | Zero-centered but saturates at ±1 |
| ReLU | $\max(0,x)$ | 1 if $x>0$, 0 else | Dead neurons when $z \leq 0$ always |
| LeakyReLU | $\max(0.01x, x)$ | 1 if $x>0$, 0.01 else | Fixes dead neurons |

### Results

Sigmoid achieves ~11% accuracy (random chance for 10 classes), confirming vanishing gradients: layer 0 receives $250\times$ weaker signal than layer 3. ReLU and LeakyReLU both reach ~97%, but LeakyReLU has ~0% dead neurons vs ~3% for ReLU.

### Dead Neuron Analysis

I implemented `compute_dead_neurons()`: passes batches with `store_activations=True`, accumulates absolute activation sums per neuron, and classifies neurons with zero total sum as dead. Adam+ReLU shows ~41% dead neurons because Adam's aggressive early updates push neurons into permanent negative territory.

### Activation Distribution Analysis

The 4×4 histogram grid reveals the mechanism behind each activation's behavior:
- **Sigmoid:** Mass at 0 and 1 → $\sigma'(x) \approx 0$ everywhere → vanishing gradient
- **Tanh:** Mass at ±1 but zero-centered → slightly better gradient flow
- **ReLU:** Large spike at exactly 0 (dead/inactive neurons), free spread for positives
- **LeakyReLU:** Small bump below 0 (the 0.01x leak), no spike at 0, healthiest overall
